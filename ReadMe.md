# DDCS

DDCS，全称Docker Desktop Chinese Script，即Docker汉化脚本。

## 说明
汉化过程仅需13s（20颗cpu核心）即可完成（具体耗时以运行结果为准）。

运行以下代码即可查看核心数：
```python
import multiprocessing
cpu_count = multiprocessing.cpu_count()
print(cpu_count)
```

## 使用方法
下载源码，管理员权限启动终端并进入到源码根目录，使用以下命令即可：
```bash
python ddcs.py
```

注意：请务必使用管理员权限启动终端。
